## 介绍

由于学习nodejs，发现了hexo这个用nodejs写的博客构建系统，看了一些例子觉得非常不错，于是就尝试用hexo来搭建了这个博客。

其中,master分支放的是博客源码，gh-pages存放的是构建之后的静态文件，使用Travis CI进行持续构建。
